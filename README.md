# Biblioo ㈢

O **Biblioo** é um sistema de gerenciamento de biblioteca desenvolvido com o objetivo de facilitar o controle de livros, usuários e empréstimos.
A aplicação permite organizar informações de forma simples e eficiente, simulando o funcionamento básico de uma biblioteca, sendo ideal para fins acadêmicos e aprendizado de banco de dados e desenvolvimento de sistemas.

---
## Funcionalidades

* Cadastro de livros
* Cadastro de usuários
* Registro de empréstimos e devoluções
* Consulta de dados / Acesso concorrente
* Organização das informações da biblioteca

---

## Tecnologias 

* Frontend: TypeScript + JSX + Tailwind CSS
* Backend: JavaScript + Express.js
* Banco de Dados: MySQL
* Controle de versão: Git
* Hospedagem do código: GitHub
